Pycharm运行tensorflow找不到libcublas.so.10.0

Posted on 2019-05-31 | In 2019

问题

在bash中可以运行Python来正确导入tensorflow,在pycharm中运行提示错误

ImportError: libcublas.so.10.0: cannot open shared object file: No such file or directory

解决

参考: Pycharm environment different than command line

直接从命令行启动Pycharm即可

原因

Pycharm无法找到LD_LIBRARY_PATH路径,即使在运行的地方手动加上这个路径也不行
