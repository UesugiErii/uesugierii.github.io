Install tuxclocker

发表于 2019-10-14 | 更新于 2020-01-08 | 分类于 2019

Only discuss the situation under ubuntu and nvidia

Requirements

nvidia-smi
nvidia-settings

The above two means that the nvidia driver has been installed.

Ubuntu

sudo apt install libxnvctrl-dev

sudo apt install qt5-default

sudo apt install libqt5x11extras5-dev

ln -s /usr/lib/x86_64-linux-gnu/libnvidia-ml.so.1 /usr/lib/x86_64-linux-gnu/libnvidia-ml.so
