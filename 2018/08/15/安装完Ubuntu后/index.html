proxychains-ng

这个不需要像网上说的那样编译安装,直接apt-get就行了,测试的时候不要用curl ip.cn这个方法,可以用
proxychains4 wget https://geoiptool.com/zh/

Gnome Shell

如何安装Gnome Shell扩展

推荐几款插件:

  • Dash to Dock
  • Hide Top Bar
  • OpenWeather
  • System-monitor

conky

Light-weight system monitor for X.

可以非常详细的看到系统各个数据,上面的System-monitor主要用来看cpu温度

ram disk

一般人应该用不到(一般人也不会用linux吧…),可以在/etc/fstab添加类似下列语句
tmpfs /home/用户名/.cache tmpfs size=2G,defaults,noatime,mode=777 0 0
ram disk随机性能读写非常快.
